**usage**

- windows only for now :(
- drop into game folder. launch vanilla through steam/game exe, and launch modded through KMM.

**features**
- enable / disable mods
- "Start Game" button launches modded; bepinex is disabled when the window is closed

**assumptions**
- game has 1 exe, in same folder as KMM.
- every folder in `BepInEx/plugins` has 1+ `.dlls`; mod dlls aren't located anywhere else.

**potential features**
- global instance / game selector
- mod icons
- reduce exe size
- install/remove mod button
